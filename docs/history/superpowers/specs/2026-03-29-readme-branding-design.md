# README Branding Design

**Goal**

把 `MedAutoScience` 的 GitHub 首页从纯文字说明，升级成更像成熟医学研究平台项目首页的展示面。

**Direction**

- 定位：医学研究平台
- 主语言：中文
- 气质：医学科技感
- 色彩：青蓝系
- 首页主标题：`MedAutoScience 医学自动科研平台`
- 副标语：`从专病数据到投稿交付的自动科研平台`

**Visual Surface**

- 顶部使用仓库内 `SVG` Logo
- 顶部加入一张横向总览主示意图
- 示意图结构：
  - 左侧：医学研究者、专病数据
  - 中间：私有数据资产、公开数据、DeepScientist、ToolUniverse、Codex 协同层
  - 右侧：分析结果、图表、稿件、投稿交付

**README Structure**

1. 顶部标题区
2. 主示意图区
3. 平台解决什么问题
4. 平台核心能力
5. 默认优先研究模式
6. 数据资产层
7. 最小部署
8. 仓库文档

**Constraints**

- 不做企业宣传页风格
- 不做纯工程框架风格
- 不暴露本地路径
- 所有视觉资产落仓库，优先使用 `SVG`
