## Archive Classes

- Archived records：closeout、completion package、旧 board，以及解释早期工作如何收口的 landed program records。
- Dated snapshots：recurring intake outputs 和周期性 learning records，用于保存某一日期已完成轮次的状态。
- Retired material：旧计划、旧过程草稿、已被替代的 workflow notes 和不再 active 的 roadmap。
- Provenance：source-intake、命名、定位和迁移痕迹，用于解释当前 contract 的来由。

## Lifecycle Rule

历史文档可作为 provenance 引用，但不覆盖 active truth。若历史记录指向旧路径或旧 contract，
执行或投稿决策前应先回到当前文档索引、runtime contracts、policy docs、references index
和 active program board。
