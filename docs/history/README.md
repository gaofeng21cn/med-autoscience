# History Archive

**English** | [中文](./README.zh-CN.md)

This directory is the archive index for repo-tracked historical material in
`Med Auto Science`.

Use this layer for archived records, dated snapshots, retired material, and
provenance that remains useful for audits or later reconstruction. It is
read-only context, not the active runtime, publication, controller, or program
execution authority.

## Archive Classes

- Archived records: closeouts, completion packages, old boards, and landed
  program records that explain how earlier work concluded.
- Dated snapshots: recurring intake outputs and periodic learning records that
  preserve a completed round at a specific date.
- Retired material: old plans, old process drafts, superseded workflow notes,
  and no-longer-active roadmaps.
- Provenance: source-intake, naming, positioning, and migration traces that help
  explain why current contracts look the way they do.

## Lifecycle Rule

Historical docs may be cited as provenance, but they do not override active
truth. If a historical record points at an older path or contract, follow the
current docs guide, runtime contracts, policy docs, references index, and active
program board before making execution or publication decisions.
