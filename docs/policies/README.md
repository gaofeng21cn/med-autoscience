# Policies

这里存放 `MedAutoScience` 已经稳定、适合长期仓库跟踪的规则文档。

它们默认服务于内部技术同事与 Agent 的稳定操作面，不默认进入首页双语公开面。
如果未来某份规则要提升到对外公开面，必须同步补齐英文 `.md` 与中文 `.zh-CN.md` 镜像。

与 `docs/` 其他说明文档的区别：

- `docs/*.md` 解释“怎么理解、怎么使用”
- `docs/policies/*.md` 规定“什么是正式边界、什么不能退回去”
