# Citation Locator Audit

Purpose: verify whether cited sources support the exact manuscript claims they are attached to. This is MAS-owned companion content, not a runtime citation checker or external reference authority.

MAS authority remains in citation ledgers, claim-evidence maps, review ledgers, publication evaluation, and controller decisions. Use this template progress-first: audit claim-critical citations first, downgrade or route unsupported claims, and record missing route-required locators explicitly. A missing locator is not a default hard gate unless the current owner route requires it.
