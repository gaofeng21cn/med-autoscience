## medical intake-audit gate

Treat the intake audit as a publication-path checkpoint, not only a file inventory step.

{{MED_AUTOSCIENCE_ROUTE_BIAS}}

{{MED_AUTOSCIENCE_STUDY_ARCHETYPES}}

At minimum, trust-rank and externalize:

- center, cohort window, study design, and ethics status
- endpoint provenance and exact outcome definition
- inclusion / exclusion criteria, variable definitions, and missing-data handling status
- current data-release lineage and whether a newer private release or public-data extension already exists
- whether the strongest current line still has a credible path to a Q2+ medical paper

If the current line is clinically thin, weak, or mostly surviving on inertia, prefer branch / reset / route reselection over polishing weak legacy outputs.
