# Medical Research Execution Skill Policy

MAS stage executors use this skill policy to perform medical research work inside a stage packet. The executor may inspect source refs, study charter, evidence ledgers, review ledgers, literature refs, runtime event refs, and publication-route memory refs. It produces artifact/source/evidence refs, execution receipts, route-back reasons, and typed blockers.

The skill does not own study truth, publication verdict, artifact authority, memory body acceptance, source readiness verdict, or submission readiness. Those outcomes require MAS owner surfaces and, where quality is involved, an independent reviewer or auditor record.
