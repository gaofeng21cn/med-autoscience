# Publication Route Memory Knowledge

Publication-route memory is MAS-owned reusable research experience. It can contain route patterns, failed paths, evidence package expectations, journal-neighbor lessons, and figure/table strategies. It is injected as a small set of refs into stage knowledge packets and written back through typed closeout proposals plus router receipts.

Memory is advisory. It cannot promote a claim to evidence, authorize publication quality, replace literature search, or decide the route without stage output and controller review. OPL may consume memory locator refs and receipt metadata only.
