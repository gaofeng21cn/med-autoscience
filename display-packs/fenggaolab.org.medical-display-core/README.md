# fenggaolab.org.medical-display-core

Built-in core pack for audited medical display templates.

## Scope

- Evidence figures registered in `src/med_autoscience/display_registry.py`
- Illustration shells registered in `src/med_autoscience/display_registry.py`
- Table shells registered in `src/med_autoscience/display_registry.py`

## Manifest bootstrap

Template manifests under `templates/*/template.toml` are generated by:

`med_autoscience.display_pack_bootstrap.export_core_pack_template_manifests(...)`
